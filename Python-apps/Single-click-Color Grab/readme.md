### v.1.1
- Added UI and Fixed Color Picker.
- Added Enter/Space to pick color.

### v.1.0
- Written in python
- Click to pick color from anywhere on the screen and the Hex code of the color is copied in the clipboard.
- Auto exits

---
### Install as .exe
```

pyinstaller --noconsole --onefile --icon="cg.ico" ColorGrab.py

```
---
### Run directly
```
pip install PyQt6
python ColorGrab.py
```
