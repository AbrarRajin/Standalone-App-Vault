# Single Click Color Grab
<img src="/Python-apps/Single-click-Color Grab/cg.ico">  


> ### v.1.0
- Written in python
- Click to pick color from anywhere on the screen and the Hex code of the color is copied in the clipboard.
- Auto exits


> ### Instal as .exe
```

pyinstaller --noconsole --onefile --icon="cg.ico" ColorGrab.py

```

> ### Run directly
```
pip install PyQt6
python ColorGrab.py
```
